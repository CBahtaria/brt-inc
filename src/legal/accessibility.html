Accessibility

Accessibility

BRT Inc. is committed to making this website usable by as many people as possible. We target WCAG 2.1 Level AA conformance.

Last updated: 2 July 2026.

Measures we take

  • Semantic HTML — correct heading hierarchy, landmark regions, and ARIA roles throughout.
  • Full keyboard navigation — every interactive element (links, buttons, form fields) is reachable and operable without a mouse.
  • Screen reader support — all images carry descriptive alt text, all form fields have associated labels, and dynamic content updates use ARIA live regions.
  • Colour contrast — text meets the WCAG AA minimum: 4.5:1 ratio for normal-size text and 3:1 for large text.
  • Reduced motion — all animations and transitions respect the prefers-reduced-motion media query and are disabled when the user has indicated a preference.
  • Skip link — a "Skip to content" link is the first focusable element on every page, allowing keyboard users to bypass repeated navigation.

Known limitations

  • Portfolio card background images are decorative and carry descriptive alt text, but complex visual layouts (multi-column grids with overlapping elements) may not convey full spatial detail to screen reader users.

Feedback

If you encounter an accessibility barrier, please let us know. Email accessibility@brtinc.dev and we aim to respond within 2 business days.
